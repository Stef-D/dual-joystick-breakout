dual-joystick-breakout
======================
Dual PS2-style joystick breakout.

Based on Sparkfun.com 's breakout board (https://www.sparkfun.com/products/9110), so all credits to them.

1 -> VCC
2 -> Vertical - left
3 -> Horizontal - left
4 -> Select/push - left
5 -> GND
6 -> Vertical - right
7 -> Horizontal - right
8 -> Select/push - right
